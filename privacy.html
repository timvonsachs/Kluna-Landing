Datenschutzrichtlinie

Kluna – Das erste Tagebuch das hört wie es dir wirklich geht.

Deine Privatsphäre ist uns wichtig. Kluna wurde von Anfang an so gebaut, dass deine persönlichsten Daten – deine Stimme – geschützt bleiben. Diese Richtlinie erklärt transparent, welche Daten erhoben werden, wie sie verarbeitet werden und welche Kontrolle du hast.

1. Stimmanalyse

Deine Stimme verlässt nie dein Gerät.

Die Analyse deiner Stimme (Tonhöhe, Tempo, Stabilität, Wärme, Pausen) passiert vollständig auf deinem iPhone. Keine Audioaufnahme wird an einen Server gesendet. Die Rohdaten deiner Stimme bleiben lokal auf deinem Gerät gespeichert.

2. Transkription

Deine gesprochenen Worte werden in Text umgewandelt, damit du deine Einträge lesen kannst. Die Transkription nutzt Apples On-Device Speech Recognition und wird lokal verarbeitet.

3. KI-Coach

Um dir persönliches Feedback zu geben, werden folgende Daten an die Anthropic API (Claude) gesendet:

• Dein transkribierter Text

• Berechnete Stimmwerte (Zahlen wie Energie, Wärme, Stabilität – keine Audiodaten)

• Deine letzten Einträge (für Kontext)

Es werden keine Audiodaten, keine Stimme und keine biometrischen Rohdaten an die API gesendet. Die Anthropic API speichert keine Nutzerdaten und verwendet sie nicht zum Training.

4. Welche Daten lokal gespeichert werden

Auf deinem Gerät speichert Kluna:

• Audioaufnahmen deiner Einträge

• Transkripte

• Berechnete Stimmwerte und Dimensionen

• Persönliche Baselines (dein individuelles Stimmprofil)

• Stimmungs-Labels und Coach-Texte

• Dein Name, Altersgruppe und Geschlecht

Alle diese Daten verbleiben auf deinem Gerät und werden nicht an Dritte weitergegeben, es sei denn, du entscheidest dich freiwillig für die Datenspende.

5. Datenspende (freiwillig)

Du kannst freiwillig anonymisierte Stimmdaten für die Forschung spenden. Diese Funktion ist standardmäßig deaktiviert und kann jederzeit ein- und ausgeschaltet werden.

Was bei einer Spende gesendet wird:

• 5 berechnete Dimensionswerte (Wärme, Stabilität, Energie, Tempo, Offenheit)

• Stimmungs-Label (z.B. „zufrieden")

• Altersgruppe und Geschlecht

Was NICHT gesendet wird:

• Keine Stimme, keine Audioaufnahmen

• Kein Text, keine Transkripte

• Kein Name, keine E-Mail, keine Geräte-ID

• Keine Möglichkeit, die Daten einer Person zuzuordnen

Die gespendeten Daten dienen dem Aufbau eines anonymen Stimm-Biomarker-Datensatzes zur Verbesserung der Erkennung emotionaler Muster.

6. Deine Kontrolle

Du hast jederzeit die volle Kontrolle über deine Daten:

• Daten löschen: In der App unter Profil → „Alle Daten löschen" werden sämtliche lokalen Daten unwiderruflich entfernt.

• Datenspende beenden: Jederzeit unter Profil → Stimmdaten spenden deaktivieren.

• Daten exportieren: Du kannst deine Daten unter Profil → „Daten exportieren" herunterladen.

• App löschen: Beim Deinstallieren der App werden alle lokalen Daten automatisch entfernt.

7. Drittanbieter

Kluna nutzt folgende Drittanbieter-Dienste:

• Anthropic (Claude API) – für KI-gestütztes Feedback. Es werden nur Textdaten und berechnete Zahlenwerte gesendet. Anthropic Datenschutz

• Apple (Speech Recognition) – für On-Device-Transkription. Daten verlassen das Gerät nicht.

Kluna nutzt keine Tracking-Dienste, keine Werbung und keine Analytics-Tools von Drittanbietern.

8. Kinder

Kluna ist nicht für Personen unter 16 Jahren bestimmt. Wir erheben wissentlich keine Daten von Kindern.

9. Änderungen

Diese Datenschutzrichtlinie kann aktualisiert werden. Wesentliche Änderungen werden innerhalb der App kommuniziert. Die aktuelle Version ist immer unter dieser URL verfügbar.
